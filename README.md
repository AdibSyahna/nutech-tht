# Nutech Take-Home-Test
## By: M. Adib Mangaraja

### Description
Projek ini dibuat untuk memenuhi tes backend programmer, dengan mengikuti requirements yang telah ditetapkan didalam `Assignment API Programmer`

### Tech Stack:
- ExpressJS
- SQLite 3

### Requirements:
- Node.js v24+

### Installation
1. Download repository ini dalam bentuk .zip
2. Ekstrak file .zip
3. Buka terminal didalam folder dimana .zip di ekstrak
4. Jalankan `npm install` untuk mengunduh modules yang dibutuhkan
5. Untuk produksi: `npm start`, untuk development: `npm test`
